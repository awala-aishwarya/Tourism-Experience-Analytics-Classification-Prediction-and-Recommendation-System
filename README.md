# 🌍 Tourism Experience Analytics  
### Classification, Prediction & Recommendation System

---

## 📌 Project Overview

This project focuses on analyzing tourism experience data to understand visitor behavior, satisfaction levels, and travel patterns.

The system performs:
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Classification
- Prediction
- Recommendation

The goal is to extract meaningful insights and build intelligent machine learning models to improve tourism experience and satisfaction.

---

## 🎯 Objectives

- Analyze rating distribution of tourists
- Study visit mode distribution (Solo, Family, Group, Couple, etc.)
- Perform data cleaning and preprocessing
- Encode categorical variables for machine learning
- Build classification models to predict tourist satisfaction
- Evaluate model performance using standard metrics
- Generate insights and recommendations based on data patterns

---

## 📊 Dataset Description

The dataset contains tourism-related information such as:

- Visitor demographics
- Visit mode
- Ratings
- Location
- Travel season
- Experience feedback
- Other tourism-related attributes

### Data Preprocessing Steps:

- Handling missing values
- Encoding categorical variables
- Feature selection
- Train-test split
- Scaling (if required)

## 📈 Key Insights

- Majority of tourists provided 4 and 5 star ratings
- Group travel is the most preferred visit mode
- Seasonal variations impact tourist satisfaction

## 🚀 How to Run the Project

1. Clone the repository:

   git clone https://github.com/awala-aishwarya/Tourism-Experience-Analytics.git

2. Navigate to the project folder:

   cd Tourism-Experience-Analytics

3. Install required libraries:

   pip install -r requirements.txt

4. Open Jupyter Notebook:

   jupyter notebook

5. Run the file:

   tourism_analysis.ipynb
